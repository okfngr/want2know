Πολιτική Προστασίας Προσωπικών Δεδομένων:

Το «Θέλω να ξέρω» είναι ένας μη-κερδοσκοπικός οργανισμός. Η ιστοσελίδα δημιουργήθηκε από φοιτητές του Αριστοτελείου Πανεπιστημίου Θεσσαλονίκης, στα πλαίσια μίας εργασίας.


Περίληψη:

Τα δεδομένα σχετικά με εσάς που συλλέγει το «Θέλω να ξέρω» είναι τα ίδια που συλλέγουν οι περισσότερες ιστοσελίδες. Η πολιτική αυτή περιγράφει την προσέγγισή μας στα προσωπικά σας δεδομένα.

Γενικά, χρησιμοποιούμε τις προσωπικές σας πληροφορίες για τον σκοπό που μας τις παρέχετε. Επίσης, χρησιμοποιούμε τα δεδομένα που συλλέγουμε από την ιστοσελίδα για να καταλάβουμε πώς τη χρησιμοποιείτε, με σκοπό να τη βελτιώνουμε συνεχώς και να σας προσφέρουμε νέες υπηρεσίες.

Θα πρέπει να γνωρίζετε πως οτιδήποτε δημοσιεύετε στην ιστοσελίδα μας, όπως ερωτήσεις, απαντήσεις, σχόλια, σύνδεσμοι κλπ θα παραμένουν για πάντα εκεί και θα αποτελούν κομμάτι αυτής, εκτός από ελάχιστες περιπτώσεις. Οπότε σκεφτείτε πριν το δημοσιεύσετε.

Σύμφωνα με τον Νόμο περί Προστασίας Δεδομένων, μπορείτε ανά πάσα στιγμή να ζητήσετε αντίγραφο των προσωπικών σας πληροφοριών που έχουμε (για το οποίο ενδέχεται να χρεωθείτε ένα μικρό ποσό για την αντιστάθμιση του κόστους διοίκησης μας) απλά γράφοντάς μας (βλ. παρακάτω).

Αυτή η πολιτική απορρήτου θα επανεξεταστεί και ενδέχεται να αναθεωρηθεί, από στιγμή σε στιγμή. Οπότε, μπορείτε να ανατρέχετε συχνά σε αυτούς τους όρους.


Πληροφορίες Υποστήριξης:

Εάν ενταχθήκατε στο δίκτυό μας, θα χρησιμοποιήσουμε τις πληροφορίες που μας δίνετε για να σας συνδέσουμε στα σωστά μέρη του δικτύου μας.

Εάν εγγραφήκατε ως μέλος της κοινότητάς μας,(σας ευχαριστούμε που το κάνατε) εκτιμούμε την υποστήριξή σας. Ως μέλος της κοινότητας, θα μπορείτε να συμμετέχετε πιο ενεργά στις δραστηριότητές μας. Για να σας βοηθήσουμε να το κάνετε αυτό, θα χρησιμοποιήσουμε τα στοιχεία που μας παρέχετε. Ενδέχεται να χρησιμοποιήσουμε τα στοιχεία επικοινωνίας σας για να έρθουμε σε επαφή μαζί σας στο πλαίσιο της εν λόγω συμμετοχής.


Δημόσια Δεδομένα και Δημοσίευση:

Η περιήγηση σας στην παρούσα ιστοσελίδα δεν αποκαλύπτει την ταυτότητά σας δημόσια. Για περισσότερες πληροφορίες δείτε την παράγραφο Ιδιωτικό Αρχείο παρακάτω σε αυτό το έγγραφο.


Ταυτοποίηση των συγγραφέων:

Όταν πάτε να κάνετε ανάρτηση μίας ερώτησης ή ενός σχολίου, σας ζητείτε ένα όνομα και μία διεύθυνση ηλεκτρονικού ταχυδρομείου(e-mail). Δε χρειάζεται να δώσετε το πραγματικό σας όνομα και το κανονικό σας e-mail. Η διεύθυνση ηλεκτρονικού ταχυδρομείου σας δε θα δημοσιευθεί. Εάν ακόμη ανησυχείτε γι’αυτό μπορείτε να φτιάξετε έναν νέο λογαριασμό ηλεκτρονικού ταχυδρομείου εντελώς δωρεάν.

Οποιαδήποτε ενέργεια στην ιστοσελίδα μας μπορεί να ταυτοποιηθεί από τη διεύθυνση IP του χρήστη. Οι αριθμοί αυτοί μπορούν να ταυτοποιήσουν τις προσωπικές σας πληροφορίες ,είτε πρόκειται για την υπηρεσία παροχής internet του σπιτιού σας ή του πανεπιστημίου σας ή του χώρου εργασίας σας. Η διεύθυνση IP σας μπορεί να χρησιμοποιηθεί σε συνδυασμό με άλλα στοιχεία για να σας εντοπίσει.

Εάν ανησυχείτε για τις προσπάθειες εντοπισμού των στοιχείων σας μέσω της IP σας, μπορείτε να χρησιμοποιήσετε μια ανώνυμη υπηρεσία περιήγησης ή να επιχειρήσετε κάποια μέσα για να συσκοτίσουν την πραγματική διεύθυνση IP σας.

Αν ισχύει κάτι τέτοιο, τότε δοκιμάστε το Tor, μια ανώνυμη υπηρεσία περιήγησης. Παρακαλώ δείτε στο http://tor.eff.org/ για περισσότερες λεπτομέρειες.


Διεύθυνση Ηλεκτρονικού Ταχυδρομείου(e-mail):

Εάν μας έχετε δώσει το e-mail σας, συμφωνείτε με το να στέλνουμε e-mail σε αυτή τη διεύθυνση, ώστε να σας παρέχουμε τις υπηρεσίες μας. Στην πράξη, αυτά τα e-mail αφορούν συνήθως σημαντικές αλλαγές σε υπηρεσίες που μπορείτε να εγγραφείτε.

Εκτός από τα απαραίτητα e-mail που θα σας αποστέλλονται, μπορεί να επικοινωνούμε μαζί σας και για δικούς μας σκοπούς. Για παράδειγμα, μπορεί να σας ενημερώνουμε για νέες υπηρεσίες που σας προσφέρουμε. Κατανοούμε πως μπορεί να είναι ενοχλητικό το να λαμβάνεται συνεχώς ανεπιθύμητα e-mail. Θα προσπαθήσουμε να σας στέλνουμε μόνο εκείνα τα μηνύματα που είναι πιο κοντά στα ενδιαφέροντά σας.


Cookies:

Η ιστοσελίδα μας χρησιμοποιεί συχνά cookies. Αυτό γίνεται για να ενεργοποιηθούν ορισμένες δυνατότητες της ιστοσελίδας. Αν θέλετε να μην αποκαλύψετε τον προσδιορισμό των πληροφοριών σας, ειδικά αν χρησιμοποιείτε ένα δημόσιο ή κοινόχρηστο υπολογιστή, τότε μπορείτε να καθαρίσετε αυτά τα cookies και την προσωρινή μνήμη του browser. Επίσης, μπορείτε να απενεργοποιήσετε τον browser σας από την αποδοχή cookies.


Καταγραφή προσωπικών στοιχείων:

Κάθε φορά που επισκέπτεστε μια σελίδα στο διαδίκτυο, μπορεί να στέλνετε κάποιες πληροφορίες στο διακομιστή. Οι webservers που φιλοξενούν αυτό το site διατηρούν αρχεία πρόσβασης με τα στοιχεία που στέλνετε. Αυτές οι πληροφορίες χρησιμοποιούνται για να παρέχουν τα στατιστικά στοιχεία της ιστοσελίδας και να αναδείξουν τις πιο δημοφιλείς σελίδες. Εμείς δε σκοπεύουμε να χρησιμοποιήσουμε αυτά τα αρχεία καταγραφής για τον εντοπισμό των νόμιμων χρηστών.

Γενικά, τα δεδομένα που καταγράφονται τα χρησιμοποιούμε για να επιλύουμε τεχνικά προβλήματα της ιστοσελίδας. Ακόμη, διερευνούμε τις περιπτώσεις κατάχρησής της. Επίσης, χρησιμοποιούμε κάποιες υπηρεσίες ανάλυσης των στατιστικών του διαδικτύου για να σας δώσουμε μία γεύση της επισκεψιμότητας της ιστοσελίδας μας.

Και πάλι, αν ανησυχείτε για την καταγραφή των προσωπικών σας δεδομένων, μπορείτε να επιλέξετε να συνδεθείτε μέσω μίας ανώνυμης υπηρεσίας περιήγησης, ώστε να κρύψετε την IP σας.


Δε γίνεται πώληση των πληροφοριών σας:

Οι προσωπικές σας πληροφορίες που συλλέγονται μέσω αυτής της ιστοσελίδας, πχ η διεύθυνση ηλεκτρονικού ταχυδρομείου δεν πωλούνται.


Ασφάλεια των προσωπικών σας πληροφοριών:

Δεν εγγυόμαστε την ασφάλεια των πληροφοριών που μας παρέχετε.
